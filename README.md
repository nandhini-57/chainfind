# 🔗 ChainFind

ChainFind is a Lost & Found web application that helps users report lost items and discover found items easily.

---

## 🚀 Features

- 🔍 Search for lost items
- 📦 Report found items
- 👤 User-friendly interface
- ⚡ Fast and responsive design
- 🔐 Secure backend handling

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML, CSS, JavaScript

### Backend
- Node.js
- Express.js / Firebase (based on your setup)

---

## 📁 Project Structure

chainfind/
│
├── frontend/ # React frontend
├── backend/ # Server / API
├── .gitignore
└── run.bat

---
Install dependencies
  Frontend
          cd frontend
          npm install
          npm run dev
  Backend
          cd backend
          npm install
          node index.js
