# 🔗 ChainFind

ChainFind is a **Lost & Found web application** that helps users report lost items and discover found items easily.

---

## 🚀 Features

* 🔍 Search for lost items
* 📦 Report found items
* 👤 User-friendly interface
* ⚡ Fast and responsive design
* 🔐 Secure backend handling

---

## 🛠️ Tech Stack

### 🎨 Frontend

* React.js
* HTML
* CSS
* JavaScript

### ⚙️ Backend

* Node.js
* Express.js / Firebase *(based on setup)*

---

## 📁 Project Structure

```
chainfind/
│
├── frontend/        # React frontend
├── backend/         # Server / API
├── .gitignore
└── run.bat
```

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/nandhini-57/chainfind.git
cd chainfind
```

---

### 2️⃣ Install Dependencies & Run

#### ▶️ Frontend

```
cd frontend
npm install
npm run dev
```

---

#### ▶️ Backend

```
cd backend
npm install
node index.js
```
